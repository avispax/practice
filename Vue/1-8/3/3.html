キー修飾子

エンターキーにのみ反応するとかできるよ。
KeyboardEvent.key ってサイトにある表はケバブ-ケースにすることで使えるよ。例はPageDown(page-down)。

キーコード

非推奨だけど、keycodeっていうコード値で同じことができるよ。enterは13だよ。

システム修飾子キー

↑に続けてさらに「ctrl」「alt」「shift」とか行けるよ。「meta」はmacのコマンド、winの田だよ。
Alt + c
ctrlを押しながらclick。
中のコードを入れ替えると動かぬという。これは公式ページのnoteにある通り、keyupとかと組み合わせて工夫が必要とのこと。

.exact 修飾子

より正確に判定します。↑のctrl + clickは、他のキーが押されていても反応していました。
.exactは対象の2つ（ctrl+click）のみとします。
とはいえ、他のキーとは「shift」とか名のあるキーのみで、「a」とか「c」みたいなのは素通りします。
