値のバインド

前段

v-modelを使うと、基本的にはstaticな文字列やtrue/falseだよね。
でも v-bind を使って工夫するとアレコレできるよ。

例１ : フツーにv-modelのみの場合

ラジオとチェックボックスとセレクト

v-modelの値を反映する : {{ radio }}, {{ checked }}, {{ selected }}

値をバインドして、v-modelの中身にv-bindした値をツッコむ。

checkbox

value : {{ check2 }}。check2の中身はyes/noの文字が入る。

radio

value : {{ radio2 }}。radio2の中身はv-bind:value="b"で定義したb1b1b1b1～みたいのを入れるなど。

select

value: {{ select2 }}。option:valueのkey:value（number:123）をつっこむ。

修飾子

.lazy

value : {{ msg1 }}。inputじゃなくてchangeイベントに反応するよ（エンター押すとか）

忘れた人ように普通の↓

value : {{ msgx }}

.number

value : {{ msg2 }}。たまに文字列を返すみたいな挙動もするブラウザがあったりとか。

.trim

value : {{ msg3 }}
